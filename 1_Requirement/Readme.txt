●必要なMATLABライセンスとバージョン

このサンプルモデルは次のバージョンを使用して作成されています。
  MATLAB R2015b

・SimulinkモデルからHDLコード生成を行うのに必要なライセンス
　必須：MATLAB, Simulink, Fixed-Point Designer, MATLAB Coder, HDL Coder
　推奨： Signal Processing Toolbox, DSP System Toolbox,

・MATLABファイルからHDLコード生成を行うのに必要なライセンス
　必須：MATLAB, Fixed-Point Designer, MATLAB Coder, HDL Coder
　推奨： Signal Processing Toolbox, DSP System Toolbox

・HDLコード生成レポートのモデルのWeb表示に必要なライセンス
　MATLAB Report Generator, Simulink Report Generator

・アプリケーションモデル（通信や画像処理モデル）で必要なライセンス
　Communications System Toolbox, Computer Vision System Toolbox, Image Processing Toolbox
  Vision HDL Toolbox
・Stateflowチャート作成に必要なライセンス
　Stateflow
・モデル検証で必要なライセンス
　Simulink Verification and Validation, Simulink Design Verifier
・コ（協調）シミュレーションによるHDL/FPGAの検証で必要なライセンス
　HDL Verifier


・コシミュレーションで必要な3rd partyツール
　ModelSim PE/SE/DE, Questa, IncisiveなどのHDLシミュレータ

・FPGA-in-the-Loop, HDLワークフローアドバイザで必要な3rd Partyツール
　Altera QuartusIIまたはXilinx ISE/VivadoなどのFPGA論理合成/配置配線ツールと対応ボード
